一、数据库操作
创建数据库：CREATE DATABASE websecurity;
查看数据库：SHOW databases;
切换数据库：USE websecurity；
删除数据库：DROP DATABASE websecurity；

tips：SQL语句对大小写不敏感，分号作为语句结束
